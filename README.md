# 7T-layer-analysis

This is the whole process of layer segementation of 7T MRI, which requires a 7T T1 image (and a functional MR image)
If you only have a T1 image, you can run the step 0 - 4 and step 18 - 19 (Skip step 5 - step 17.)
If you have a functional MR image, please run the whole pipeline.
